of operator
  • of operator is used to create an observable from a string, or an array or an object
  • no processing is done, it returns list as it is
from operator
  • from operator is used to create an observable from an array or an array like object or an iterable object or an observable like object
  • it processes each element in the list
fromEvent operator
  • fromEvent operator creates an observable that emits events of specific type coming from the given event target
  • we can bind target elements and methods to make sure we get observable as ouput
  • like move click event actions as observable, simply creats an observable from an event
  • in Angular, we will use ViewChild, NativeElement as target element and bind events
interval operator
  • interval operator creates an observablethat emits sequential numbers every specified interval of time, on a specified SchedularLike
